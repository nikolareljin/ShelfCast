# Hardware

## Required

- Raspberry Pi 3 with power supply
- microSD card (8GB+)
- Nook Simple Touch or an Android tablet
- USB OTG cable for Nook (if using a USB connection)

## Notes

- The Nook Simple Touch can be used as an e-ink display over USB with community drivers.
- Touchscreen support varies by kernel/driver. Validate input before committing to a mount.
- Android tablets can be used via a browser in kiosk mode pointing to the Pi dashboard.
